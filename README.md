# 🎬 Netflix Analysis Project

## 📌 Overview

Data analysis project exploring Netflix content to uncover trends in genres, countries, and release patterns using Python and Power BI.

---

## 🛠️ Tools

* Python 
* Power BI
* Jupyter Notebook

---

## 📊 Key Insights

* Movies are more common than TV Shows
* USA and INDIA dominates content production
* Drama & Comedy are top genres
* Content growth increased significantly after 2015


---

## 📂 Project Files

* `data/` → dataset
* `python/` → analysis notebook
* `powerbi/` → dashboard file

---

## 🚀 Usage

Open the notebook for analysis and the Power BI file for dashboard exploration.
